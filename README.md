Este é um app está sobe o desenvolvimento para o trabalho de extensão da matéria "Programação para Dispositivos Móveis em Android", do curso de Análise e Desenvolvimento de Sistemas EAD Estácio.
O app está interligado com o estudo da matéria, o app tem o objetivo de ajudar pequenos comerciantes, a gerenciar seu négocio, uma maneira de iniciar comerciantes que antes utlizavam caderninhos para marcar pedidos, utilizar uma tecnologia para facilitar e guardar informações de produtos, clientes e faturamento, o app oferece cadastro de novos produtos e clientes, lista de produtos e clientes cadastrado, carrinho e pedidos de produtos e informações de faturamento mensal, situação de pagamentos dos clientes, o app foi desenvolvido em react-native com framework Expo, banco de dados foi utilizado Axios, MongoDB, Ngrok pra um banco de dados online.
